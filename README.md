# 和太鼓リズムトレーナー

スマートフォンで太鼓を叩いて譜面を登録し、16分音符・32分音符へ自動クオンタイズして練習できるWebアプリです。

## 公開ページ

https://connectmolly.github.io/wadaiko-rhythm-trainer/

iPhoneでは上記URLをSafariで開いてください。Safariの共有ボタンから「ホーム画面に追加」を選ぶと、アプリのように起動できます。

## 標準搭載曲

- 曲名：まつり2026-07-11
- BPM：100
- 小節数：4
- 分解能：32分音符

## 主な機能

- スマホ画面の太鼓をタップして譜面登録
- 登録前の4拍カウント
- 16分音符・32分音符への自動クオンタイズ
- 譜面の確認・修正
- 左面・右面・左縁・右縁・両手打ち
- 練習速度、ノーツ間隔、ノーツサイズの調整
- スコア、コンボ、ハイスコア保存
- オフラインキャッシュ対応
