練習用太鼓

A：左上の縁S：左面K：右面L：右上の縁
両面・両縁は左右を約100ms以内に叩くと同時打ちとして判定します。

使い方

  1. 曲名、BPM、小節数を設定します。
  2. 「登録開始」を押すと、必ず4拍のクリックが鳴ります。4拍の後から記録が始まります。
  3. カウント後、画面上の太鼓またはA・S・K・Lキーで演奏します。
  4. 終了すると、選択した16分音符または32分音符へ自動クオンタイズした譜面が生成されます。
  5. 保存後、「練習・スコア」から選んで反復練習します。

クオンタイズ仕様

16分音符では1拍を4分割、32分音符では1拍を8分割し、最も近い位置へ丸めます。同じ位置で左右の面、または左右の縁を約100ms以内に叩いた場合は両手打ちとして保存します。

標準搭載曲

まつり2026-07-11 をHTML内部に収録しています。別の端末で初めて開いた場合も、JSONの読み込みなしで練習できます。譜面修正後に保存した場合は、その端末では修正版が優先されます。

配布について

GitHub Pagesで公開したURLをiPhoneのSafariで開くと利用できます。ホーム画面へ追加するとアプリのように起動できます。

オリジナルの和太鼓練習用Webアプリです。特定の市販ゲームとは関係ありません。
